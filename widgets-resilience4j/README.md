# resilience
> 一个容错（fault tolerance）框架，[官网](https://github.com/resilience4j/resilience4j)

## 基本模块
* 断路器：circuit breaker
* 隔离机制：bulk head
* 限流器：rate limiter
* 重试：retry
* 限时器：time limiter
